# LeKai教案知识库 — 项目规划

> 面向福建宁德地区小学语文老师的 AI 教案生成平台：输入课题+要求 → 生成教案（Markdown）+ 教案辅导说明（"鱼+渔"）

---

## 一、核心需求（已确认）

| 维度 | 决策 |
|------|------|
| **项目名称** | LeKai教案知识库 |
| **核心功能** | 教师输入"年级+课题+要求" → DeepSeek 生成教案（Markdown）+ "好教案好在哪"的辅导说明 |
| **目标用户** | 福建宁德地区小学语文老师 |
| **教材体系** | 统编版/部编版（人教社），福建宁德使用全国统一教材 |
| **平台** | 网页应用 |
| **LLM** | DeepSeek API（已有，管够） |
| **部署** | 云服务器 |
| **知识库策略** | 路线A：预先搜集筛选优秀教案建种子库（MVP先跑通） |
| **输出** | 教案 Markdown + 辅导教案的教案 |

## 三、技术选型

| 层 | 技术 | 理由 |
|----|------|------|
| **前端** | Next.js (React) + TailwindCSS | SSR友好，轻量，Markdown渲染（react-markdown） |
| **后端** | FastAPI (Python) | 异步支持好，与LLM/RAG生态契合，DeepSeek SDK兼容 |
| **向量数据库** | ChromaDB | 轻量嵌入式，零运维，适合MVP种子库 |
| **Embedding** | 硅基流动 / DeepSeek Embedding | 中文效果好，API方便 |
| **LLM** | DeepSeek API (deepseek-chat) | 用户已有，中文能力强，便宜 |
| **文档解析** | Markdown直接编写 + pdfplumber（备用） | 种子库手工整理Markdown为主 |
| **部署** | Docker Compose (前端+Nginx+后端) | 云服务器一键部署 |

### RAG 流程详解

```
用户输入："三年级《富饶的西沙群岛》2课时，重点修辞手法"
    │
    ▼
① Query 改写：将用户需求转化为检索查询
    │
    ▼
② 向量检索：在种子库中检索 Top-K 篇相似教案
    │  (按年级+课文+教学重点多维度匹配)
    ▼
③ Prompt 组装：
    System: 你是小学语文教研员，基于优秀教案参考生成新教案...
    Context: [检索到的3-5篇优秀教案]
    User: 请为三年级《富饶的西沙群岛》生成2课时教案...
    │
    ▼
④ DeepSeek 生成：
    ├── 教案正文（教学目标、重难点、教学过程、板书设计、作业）
    └── 教案辅导说明（设计意图、为什么这样教、好教案的亮点分析）
    │
    ▼
⑤ 后处理：Markdown 格式化，输出两份文档
```

## 五、知识库设计

### 种子库结构

```
knowledge-base/
├── grade-1/       # 一年级
│   ├── 识字/
│   ├── 阅读/
│   └── 口语交际/
├── grade-2/       # 二年级
├── grade-3/       # 三年级
├── grade-4/       # 四年级
├── grade-5/       # 五年级
└── grade-6/       # 六年级
```

每篇教案 Markdown 格式：
```markdown
---
grade: 三年级
semester: 上
unit: 第六单元
lesson: 富饶的西沙群岛
type: 阅读课
class_hours: 2
tags: [修辞手法, 写景, 说明文]
source: 国家智慧教育平台
---

# 《富饶的西沙群岛》教案

## 一、教学目标
...

## 二、教学重难点
...

## 三、教学过程
...
```

### MVP 种子库目标

- 每个年级 5-10 篇标杆教案（覆盖主要课文和课型）
- 总计 30-60 篇作为 MVP 起点
- 来源：国家智慧教育平台、百度文库人工筛选

## 七、验收标准

- [ ] 输入"三年级《富饶的西沙群岛》2课时"，5分钟内生成完整教案
- [ ] 教案包含：教学目标、重难点、教学过程、板书设计、课后作业
- [ ] 同步生成"辅导说明"：说明设计意图、好教案的标准、为什么这样教
- [ ] 检索到的参考教案与目标课题**年级/课型匹配准确**
- [ ] 网页可用，移动端基本可浏览
- [ ] 生成的教案符合统编版教材体系，无知识性错误
