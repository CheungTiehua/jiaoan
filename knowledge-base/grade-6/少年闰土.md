# 《少年闰土》教案

## 教材分析
《少年闰土》节选自鲁迅的小说《故乡》，是统编版六年级上册第八单元"走近鲁迅"的第一篇课文。课文通过"我"的回忆，刻画了闰土这个海边农村少年的形象——"紫色的圆脸，头戴一顶小毡帽，颈上套一个明晃晃的银项圈"。课文以典型事件（雪地捕鸟、海边拾贝、看瓜刺猹、看跳鱼儿）表现闰土的见多识广和聪明能干。

## 学情分析
六年级学生已具备初步的文学鉴赏能力，能分析人物形象和写作手法。首次接触鲁迅作品，语言表达上有些隔膜，需给学生充分的时间品味语言。本课是学生与鲁迅的第一次"相遇"，不应求全求深。

## 教学目标
1. 会写"胯、郑"等12个字，理解"其间、家景、祭祀"等词语。
2. 正确、流利、有感情地朗读课文，背诵第1自然段。
3. 通过典型事件理解闰土的形象特点。
4. 体会"我"对闰土生活的羡慕和对自由生活的向往。
5. 初步了解鲁迅及其作品风格。

## 教学重难点
- 重点：通过典型事件把握闰土形象（见多识广、聪明能干、活泼可爱）。
- 难点：理解"院子里高墙上的四角的天空"的深层含义。

## 教学过程

### 一、走近鲁迅（5分钟）
1. 简介鲁迅：中国现代文学奠基人，《故乡》的背景。
2. 学生交流已有知识，教师补充"绍兴""闰月五行缺土"等文化常识。

### 二、初读感知（10分钟）
1. 自由读，圈不理解的词，同桌互助。
2. 指名读最打动自己的段落。
3. 梗概：课文写了闰土哪些事件？（雪地捕鸟、海边拾贝、看瓜刺猹、看跳鱼儿）

### 三、品读闰土（20分钟）
1. 外貌描写：朗读第1段"深蓝的天空中挂着一轮金黄的圆月……"，口头想象画面，体会文字的画面美（背诵）。
2. 典型事件精读：分小组各选一个事件，分析闰土的语言、动作、神态。
   - 雪地捕鸟：动词"扫→支→撒→缚→拉"的精准。
   - 看瓜刺猹："捏了胡叉""轻轻地走去"，聪明勇敢。
3. 概括闰土形象：见多识广、聪明能干、活泼可爱、朴实自然。
4. "我"的感受："啊！闰土的心里有无穷无尽的希奇的事"，体会"我"对自由生活的向往。

### 四、难点突破（5分钟）
1. "院子里高墙上的四角的天空"——在高墙大院里长大的"少爷"生活与闰土广阔天地的鲜明对比。
2. 引导学生联系实际：你生活中有没有"四角的天空"？

### 五、总结拓展（5分钟）
1. 总结写人方法：外貌+典型事件+对比。
2. "三十年后的闰土会变成怎样？"设疑，为初中学《故乡》全篇做铺垫。

## 板书设计
```
少年闰土
  外貌：紫色圆脸 小毡帽 银项圈
  事件：
    雪地捕鸟 → 聪明
    看瓜刺猹 → 勇敢
    海边拾贝 → 见多识广
    看跳鱼儿 → 活泼
  我的感受：羡慕 向往
  四角的天空 → 自由的渴望
```

## 作业布置
1. 仿照第1段，用一个画面描摹一个人物。
2. 阅读鲁迅《故乡》全文或推荐的其他鲁迅作品选段。
3. 搜集关于鲁迅的小故事，下节课分享。

## 教学反思
本课是学生与鲁迅的"第一次相遇"，教学定位为"认识闰土、感受文字美、了解鲁迅"。学生对典型事件分析到位，能准确概括闰土形象。"四角的天空"是理解难点，三分之二的学生能说出自己的理解，三分之一仍需要课后进一步点拨。重在种子，不求全求深。
