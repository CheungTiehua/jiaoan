# 《慈母情深》教案

## 教材分析
《慈母情深》是统编版五年级上册第六单元的一篇精读课文，作者梁晓声。课文记叙了母亲在极其艰难的生活条件下，省吃俭用支持"我"买书的故事，表现了深沉的母爱和"我"对母亲的感激之情。课文以"我"的情感变化为线索，通过细腻的动作、语言、神态、环境描写刻画人物。

## 学情分析
五年级学生已能通过人物的行动、语言、神态体会人物内心。对"母爱"主题有感性的生活体验，但"一元五角"的历史背景需要补充。体会作者从"想买书"到"为母亲心疼"的情感变化是理解难点。

## 教学目标
1. 认识"魄、抑"等11个生字，会写"辞、抑"等16个字。
2. 理解课文内容，体会母亲对子女无私的爱和"我"对母亲的感激之情。
3. 通过人物描写体会人物内心，学习通过细节描写表现人物情感的方法。
4. 背诵课文最后两个自然段。

## 教学重难点
- 重点：通过母亲的外貌、动作、语言描写体会人物形象。
- 难点：体会"我"从想要钱买书到为母亲心疼的情感转变。

## 教学过程

### 一、情境导入，引发共鸣（5分钟）
1. 自由交流："妈妈为你做过什么事让你觉得特别感动？"
2. 揭示课题，简介作者梁晓声及其成长背景。

### 二、初读感知，理清脉络（10分钟）
1. 自由读，圈出生字，质疑解疑。
2. 概括课文主要内容。
3. 梳理"我"的情感变化：渴望买书→被母亲工作环境震撼→心疼母亲→"鼻子一酸"。

### 三、品读细节，体会人物（20分钟）
1. 环境描写：厂房"七八十台缝纫机""光线昏暗""灯泡烤着脸"——数十个"七八十"的排比效果。
2. 母亲形象：找出描写母亲外貌、动作的句子（"极其瘦弱的脊背弯曲着""龟裂的手指""立刻"的重复使用）。
3. 语言描写："我挺高兴他爱看书的！"——抓住关键词"挺高兴""快四十岁""一元五角"体会深意。
4. "我"的情感转变：从"想买书"到"鼻子一酸""攥着钱跑了出去""给母亲一听水果罐头"。
5. 研读结尾"我想我没有权利用那钱再买任何别的东西……"——体会这句话的重量。

### 四、朗读升华（5分钟）
1. 选择最打动自己的段落，同桌互读。
2. 配乐朗读结尾两段。

### 五、写法迁移（5分钟）
1. 总结运用细节描写（外貌、动作、语言、神态）表现人物情感的写法。
2. 布置练笔。

## 板书设计
```
慈母情深
  母亲的工作环境：拥挤 闷热 嘈杂
  母亲的形象：瘦弱 疲惫 龟裂的手指
  母亲的话语："我挺高兴他爱看书的！"
  我的情感：渴望→心疼→感恩
  ← 母爱无言，如山似海 →
```

## 作业布置
1. 为妈妈做一件事，并写一段自己的感受。
2. 背诵课文最后两段。
3. 推荐阅读梁晓声《母亲》。

## 教学反思
本课通过聚焦"环境描写""人物细节""情感转变"三条线索，引导学生从语言中体会"慈母情深"。学生对"鼻子一酸""攥着钱跑了出去"等细节反应热烈，说明文本已打动了他们。补充历史背景是必要的，否则学生难以体会"一元五角"在那个年代的意义。
