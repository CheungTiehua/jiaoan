# 《秋天的雨》教案

## 教材分析
《秋天的雨》是统编版三年级上册第二单元的一篇抒情散文，以"秋雨"为线索，按照"总体引入→分述颜色、气味、声音→总结"的结构，将秋天众多景物巧妙地串联起来，描绘了一个丰收、欢乐、五彩缤纷的秋天。课文语言优美，运用大量比喻、拟人的修辞手法。

## 学情分析
三年级是习作的起步阶段，学生开始学习围绕中心意思来组织材料。本文"总分总"的结构、"五感"描写方法对于学生习作具有范例价值。

## 教学目标
1. 认识"钥、匙"等10个生字，会写"盒、颜"等13个字。
2. 正确、流利、有感情地朗读课文，背诵第2自然段。
3. 理解课文内容，感受秋天的美好，体会作者对秋天的喜爱之情。
4. 学习"总分总"结构及"五感"描写方法，仿写。

## 教学重难点
- 重点：品读第2自然段，感受颜色描写的丰富性。
- 难点：学习运用"五感"进行景物描写。

## 教学过程

### 一、情境导入（5分钟）
1. 播放秋雨视频或音频，学生谈感受。
2. 板书课题，指导"雨"字的书写。

### 二、初读感知（10分钟）
1. 自由读，读准生字，思考：课文从哪几个方面写秋天的雨？
2. 指名分段读，检查生字。
3. 梳理结构：颜色（第2段）、气味（第3段）、声音（第4段）。

### 三、品读颜色段落（15分钟）
1. 指名读第2段，圈出写颜色的词语。
2. "它把黄色给了银杏树，黄黄的叶子像一把把小扇子"——体会比喻的妙处。
3. "它把红色给了枫树，红红的枫叶像一枚枚邮票"——"一枚枚"的表达效果。
4. "金黄色是给田野的，看，田野像金色的海洋"——排比增强气势。
5. 拓展："秋天的雨还会把什么颜色给谁？"仿说练习。

### 四、品读气味和声音段落（8分钟）
1. 第3段："小朋友的脚，常被那香味勾住"——"勾"字好在哪里？
2. 第4段："吹起了金色的小喇叭"——拟人手法赏析。

### 五、总结写法，练习仿写（7分钟）
1. 总结"五感"写法：颜色（视觉）、香味（嗅觉）、声音（听觉）。
2. 仿写：用"五感"写一段关于季节的话。

## 板书设计
```
秋天的雨
  是一把钥匙（总起）
  颜色：黄色、红色、金黄色、橙红色……
  气味：香香的、甜甜的
  声音：吹起了小喇叭
  带给大地丰收，带给小朋友欢乐（总结）
```

## 作业布置
1. 背诵第2自然段。
2. 运用"五感"方法写一段关于你眼中秋天的短文。
3. 收集描写秋天的好词好句。

## 教学反思
本课重点突破了"总分总"结构和"五感"描写的读写结合点。学生对比喻、拟人修辞手法掌握较好，但仿写环节部分学生思路不够开阔。后续需提供更多范例支架，降低仿写门槛。
