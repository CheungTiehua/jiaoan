# LeKai教案系统 — 管理员操作指南

## 1. 登录管理端

访问 `http://<服务器IP>/admin`，使用管理员账号登录。

## 2. 创建教师账号

1. 进入管理端 → "用户管理"
2. 输入教师用户名和初始密码
3. 系统自动分配 `teacher` 角色

## 3. 创建教研组长账号

1. 先用 "创建教师账号" 方法创建账号
2. 进入管理端 → "角色管理"
3. 将目标用户的角色从 `teacher` 改为 `reviewer`

教研组长拥有：仪表盘查看、教案审核、知识库管理、健康检查等权限。

## 4. 上传优秀教案入库

支持 `.md`、`.docx`、`.txt` 格式：

1. 进入管理端 → "教案入库"
2. 选择教案文件（内容需超过 100 字符）
3. 系统自动用 DeepSeek 格式化为标准模板
4. 格式化后自动运行入库脚本，写入 ChromaDB 向量库
5. 入库成功后自动刷新搜索索引

限制：单文件不超过 5MB。

## 5. 查看知识库 Chunks

1. 进入管理端 → "知识库管理"
2. 可浏览所有已入库的知识点片段
3. 支持搜索、按年级筛选
4. 管理员可删除单个 chunk（删除后索引自动刷新）

## 6. 审核教师提交的教案

审核流程：

1. 教师端生成教案后 → "提交审核"
2. 管理端 → "审核队列" 出现待审核项
3. 点击查看完整教案内容
4. 可添加段落级标注（表扬/改进建议/备注）
5. **通过**：教案标记为已批准
6. **打回**：填写打回理由，教师可修改后重新提交

## 7. 导出教案

1. 在生成历史中找到目标教案
2. 点击导出按钮
3. 选择格式：Markdown (.md) 或 Word (.docx)
4. 导出的 docx 包含标准标题层级和教案辅导说明

## 8. 查看反馈和评价数据

1. 进入管理端 → "反馈统计"
2. 查看教案评分分布、平均评分
3. 查看使用频率高的标签
4. 反馈数据自动反哺检索权重

## 9. 系统维护

### 备份

管理端 → "系统备份" → 一键下载备份 zip。

### 查看设备信息

管理端 → "设备信息" 查看 MAC 地址、磁盘空间、授权状态、版本号。

### 查看系统健康

管理端 → "系统健康" 检查 API Key 有效性、Embedding 模型状态、知识库 chunk 数量。

### 提示词调优

管理端 → "提示词配置" 可在线编辑 System Prompt，保存后立即生效（无需重启）。
