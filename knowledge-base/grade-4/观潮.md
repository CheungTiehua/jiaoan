# 《观潮》教案

## 教材分析
《观潮》是统编版四年级上册第一单元的一篇写景散文，记叙了农历八月十八钱塘江大潮的壮观景象。课文按照"潮来前→潮来时→潮过后"的时间顺序，运用比喻、夸张等修辞手法，生动再现了"天下奇观"。文章层次分明，语言富有气势。

## 学情分析
四年级学生已能独立阅读叙事类文章，理解按顺序描写的写法。对钱塘江大潮缺乏直观经验，需借助视频和图片建立直观感受。

## 教学目标
1. 认识"昂、沸"等7个生字，会写"潮、称"等15个字。
2. 有感情地朗读课文，背诵第4-5自然段。
3. 了解钱塘江大潮的壮观景象，感受大自然的神奇。
4. 学习按时间顺序描写景物变化的写法。

## 教学重难点
- 重点：品读"潮来时"段落，体会语言的气势。
- 难点：学习按时间顺序描写变化，运用比喻和夸张增强表达效果。

## 教学过程

### 一、视频导入，激趣质疑（5分钟）
1. 播放钱塘江大潮视频，学生谈惊叹感受。
2. "为什么钱塘江大潮被称为天下奇观？"带着问题读文。

### 二、初读感知，理清顺序（10分钟）
1. 自由读，圈生字，读准"闷雷""沸腾""风号浪吼"。
2. 梳理写作顺序：潮来前（2段）→潮来时（3-5段）→潮过后（6段）。

### 三、品读"潮来时"，体会写法（20分钟）
1. 第3段："午后一点左右，从远处传来隆隆的响声，好像闷雷滚动"——未见其形，先闻其声。
2. 第4段：梳理描写顺序（远处→再近些→越来越近），体会由远及近的描写方法。
3. 第5段："那条白线很快地向我们移来，逐渐拉长、变粗，横贯江面"——抓动态词语。
4. 对比"白线→水墙→白色战马"三个比喻，体会大潮从小到大、从弱到强的气势变化。
5. 摘录文中比喻句，体会表达效果。

### 四、品读"潮来前"和"潮过后"（5分钟）
1. "潮来前"：人山人海——侧面衬托大潮吸引力。
2. "潮过后"："霎时"——潮水退去的速度感。

### 五、总结写法，迁移训练（5分钟）
1. 总结按时间顺序写变化的方法。
2. 模仿写一段场景（如"升旗仪式""运动会入场"）。

## 板书设计
```
观潮（天下奇观）
  潮来前：人山人海
  潮来时：
    远处→闷雷滚动
    近些→一条白线
    更近→一道水墙
    眼前→白色战马
  潮过后：风号浪吼
```

## 作业布置
1. 背诵第4、5自然段。
2. 观察一处变化的景象，按时间顺序记录下来。
3. 搜集描写长江、黄河、瀑布的经典片段。

## 教学反思
本课以"潮来时"为核心，引导学生从词语和修辞角度品味语言。视频辅助有效帮助学生建立直观感受。在仿写环节，部分学生仍停留在模式化模仿阶段，后续需加强个性化的表达训练。
